# 辅助类

-   `-Wall -Wextra` 编译时警告

-   `-Wshadow` 当内部变量名覆盖了外部变量时，产生警告。以下代码会产生警告

    ```c++
    int n;
    void solve(){
        // Solve the problem
    }
    int main(){
        int n; cin >> n;
        solve();
    }
    ```

-    ` -fsanitize=address`检查内存越界等错误（仅`Linux`？）
