## String concatenation with null

实例代码：

```java
String s1 = "abc";
String s2 = null;
System.out.println(s1 + s2); // abcnull
```

这是因为 Java 将 `s1+s2` 转化为

```java
new StringBuilder(s1).append(s2).toString()
```
