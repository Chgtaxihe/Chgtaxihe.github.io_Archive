# 一两句话小技巧



### 二进制枚举子集

```c++
int S = 11; // 1011b
for(int x=S; x>0; x=(x-1) & S) // do something
// 1. x = 1011 2. x = 1010 3. x = 1001
// 4. x = 1000 5. x = 0011 6. x = 0010
// 7. x = 0001 8. x = 0000
```
